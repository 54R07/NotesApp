__üìù NotesApp__

Note-taking application built using Isar for local database storage and Provider for state management. This app allows users to create, edit, update, and delete notes.

<hr style="border: 2px solid #ddd; margin: 30px 0;">

__Features__
 
- Create Notes: Add new notes with content

- Edit Notes: Modify existing notes

- Delete Notes: Remove unwanted notes

- Dark/Light Theme: Choose your preferred theme

- Local Storage: All data stored securely on device

- Offline Support: Works without internet connection

  <hr style="border: 2px solid #ddd; margin: 30px 0;">
  
__Setup__

Run the following commands from your terminal:

1. `git clone https://github.com/54R07/NotesApp` to clone this repository

2. `flutter pub get` in the project root directory to install all the required dependencies.
